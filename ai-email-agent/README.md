# AI Email Agent (n8n)

An AI-assisted email automation built using n8n that processes incoming emails,
classifies intent, retrieves relevant knowledge, and drafts professional responses
automatically.

---

## Overview

The AI Email Agent listens for incoming Gmail messages and uses an LLM-powered
pipeline to determine whether the email requires customer support handling.
If relevant, it retrieves internal documentation using a vector store and
drafts a response automatically.

---

## Key Concepts

- Event-driven automation
- AI-based intent classification
- Retrieval-Augmented Generation (RAG)
- Vector databases
- Human-in-the-loop email drafting
